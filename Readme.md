## Setup/Installation Requirements

* _npm install_
* _npm run build_
* _make .env file example:_ 
* _DB_NAME=postgres_
* _DB_PASSWORD=1q2w3e4r_
* _DB_PORT=5432_
* _DB_HOST=db_
* _DB_USER=postgres_
* _PORT=4040_
* _docker-compose up -d_
* _docker-compose logs -f_
